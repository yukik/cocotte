ウインドウとは

ウインドウとは、操作対象をひとつの単位としてまとめたものです。

ここでは概要だけを説明します。より詳しい内容はWindow APIを確認してください。

ウインドウを開く事で、一つの作業が開始され閉じる事で終了するように作成すると優れた業務システムになります。
ただ、一つの作業でも様々な情報を同時に参照する必要が出てきます。 その際に、作業上が不可分な情報はひとつのウインドウに設定し、参考程度の情報などは 別ウインドウに作成してください。またウインドウからウインドウへの開く為の手段を多くもうける事は結果的に作成するコンテンツ（情報）の数を減らす事が出来ます。
cocotteのウインドウは、開発コストを抑えるのに力を発揮するでしょう。

ウインドウの作成方法

ウインドウを作成するには、windowsフォルダに更にウインドウ名と同じフォルダを作成します。
その中にいくつかのファイルを配置する事でウインドウを作成します

ファイルの種類

index.js
index.jsは唯一必須のファイルです。ウインドウの設定情報はこのファイルに記載します。
一番単純な記載方法は次の通りです 
var win = {
    title   : 'ウインドウのタイトルバーの文字列です。'
  , contents: '内容です。内容はテキストではなくHTMLとして解釈されます。'
};
module.exports = exports = win;
client.js
ウインドウ毎にスクリプトを作成する場合に使用します。
単純な記載方法は次の通りです 
client.custom1 = function custom1 (e) {
    //処理内容
};
client.css
ウインドウ毎にCSSを設定する場合に使用します。
CSSはウインドウ内のコンテンツ要素のみに適用されるように設定されます。
contents {
    padding: 20px;
}
p {
    margin: 20px;
}
(任意のファイル)
コンテンツが多くなるとindex.jsに記載せずに別ファイル化したほうが管理しやすくなります。
