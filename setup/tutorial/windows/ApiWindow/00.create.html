ウインドウの作成方法は、API概要のウインドウの項で解説した通りです。
ここではより細かな設定を行ったウインドウの作成方法を解説をします。
まず必須ファイルであるindex.jsについてこの項では記載します

単純なコンテンツは次のように記載する事が出来ました。 

var win = {
    title   : 'ウインドウのタイトルバーの文字列です。'
  , contents: '内容です。内容はテキストではなくHTMLとして解釈されます。'
};
module.exports = exports = win;

contentsのプロパティは内容を示しますが、本来はテキストではなくコンテンツの配列を設定する必要があります。 先ほどの例を本来の形式で記述すると次の様になります 

var win = {
    title   : 'ウインドウのタイトルバーの文字列です。'
  , contents: [{html: '内容です。内容はテキストではなくHTMLとして解釈されます。'}]
};
module.exports = exports = win;

contentsの中身が一つだけの場合は、次のように配列でない形式で設定してもかまいません。次も同じ設定になります。 

var win = {
    title   : 'ウインドウのタイトルバーの文字列です。'
  , contents: {html: '内容です。内容はテキストではなくHTMLとして解釈されます。'}
};
module.exports = exports = win;

次の例は配列を設定する事で、４つコンテンツを表示する事の出来るウインドウです。
また、コンテンツの種類もコンテンツ毎に設定する事が出来ます。 

var win = {
    title   : 'ウインドウのタイトルバーの文字列です'
  , contents: [
        {html   : 'HTMLのコンテンツです'}
      , {text   : 'テキストのコンテンツです'}
      , {url    : 'http://cocotte.com'}  // iframeのコンテンツです
      , {portlet: 'Map'} // 地図ポートレットを表示するコンテンツです
    ]
};
module.exports = exports = win;

コンテンツの種類

全部で４種類あります。

html
HTMLとして解釈され、タグは処理されません
text
",&,<,>,半角スペースは実体参照に、タグは半角スペース4個に変換されます。 全体が<div class="textContents">（内容）</div>で 囲まれます。
url
iframeを追加します
portlet
ポートレットを追加します。 選択したポートレットによってはオプションの設定が必要になります。 各ポートレットのAPIを確認してください

コンテナ

複数のコンテンツはそのままでは、すべての文字列がただ連結されます。
containerプロパティを次のように追加してください。 適切に配置をする為の仕組みとしてコンテナを設定する事が出来ます。

var win = {
    title   : 'ウインドウのタイトルバーの文字列です'
  , contents: [{html: 'HTML'}, {text: 'テキスト'}, {url: 'cocotte.com'}, {portlet: 'map'}]
  , container: '<h2>HTMLです</h2>'
             + '<div>{0}</div>'
             + '<div>{1}</div>'
             + '<div>{2}</div>'
             + '<div>{3}</div>'
};
module.exports = exports = win;

containerに設定したHTML構文の{数字}の部分はプレースホルダーになりコンテンツに置き換えられます
