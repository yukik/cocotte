ウインドウオブジェクト

ウインドウオブジェクトとは、スクリプト内でElement.getWin()にて取得出来るオブジェクトです。
ウインドウを操作したい場合はウインドウオブジェクトのメソッドを利用してください。
例では、メソッドチェーンにより連続して操作を行っています。 

script.winOperate = function winOperate (e) {
    var win = this.getWin();
    if (win) {
        win.normal()
            .move(100, 300)
            .move()
            .setTitle('タイトル変更')
            .resize(800, 800)
            .move()
            .resize(500, 300)
            .move();
    }
};


ウインドウを取得する方法としてcocotte.Win.findメソッドを使用する方法があります。
同じビルトイン名のウインドウを検索し最初に見つかったものを返します。 

var win = cocotte.Win.find('tutorialMap');
if (win) {
    //ウインドウの操作
}

メソッド

show
最小化状態のウインドウを表示します
show()
@return メソッドチェーン
setTitle
タイトルを変更します
setTitle(title)
@param {String} title
@return メソッドチェーン
active
ウインドウを最前面に表示します active()
@return メソッドチェーン
changeStatus
ステータスのメッセージを変更します
changeStatus(message, level)
@param {String} message
@param {String} level 省略時はinfo
@return メソッドチェーン

levelはメッセージのspanエレメントにクラスを付加します
標準クラスはinfo, alert, errorです
この操作は同期対象外です
toggle
通常表示時に最大化に、最大化時に通常表示にします
toggle()
@return メソッドチェーン
normal
通常表示にします
normal()
@return メソッドチェーン
maximize
最大化します
maxiimize()
@return メソッドチェーン
minimize
最小化します
minimize()
@return メソッドチェーン
close
ウインドウを閉じます
close()
@return メソッドチェーン
showChangeBoard
チェンジボードを表示する
showChangeBoard()
@return メソッドチェーン
showHelp
ヘルプを表示する
showHelp()
@return メソッドチェーン
move
移動する。
move(top, left, calback)
@param {Number} top
@param {Number} left
@param {Function} callback ({Error} err, {Win} win)
@return メソッドチェーン

引数はすべて省略可能です。
top, leftが省略された場合は、ランダムに移動します。
resize
サイズを変更する
resize(width, height, calback)
@param {Number} width
@param {Number} height
@param {Function} callback ({Error} err, {Win} win)
@return メソッドチェーン

callbackは省略可能です。
putIn
ウインドウを使用しやすいように動作します。
putIn()
@return メソッドチェーン
modeにより動作が次のように異なります。
normal,dialog,dialog
アクティブ化、画面外にある場合は画面内に移動、画面内にある場合は微動
minimize
表示
maximize
アクティブ
el
コンテンツのDOMツリーからセレクターで指定したjQueryエレメントを取得する el(selector)
@param {String} selector @return {jQuery} elements

プロパティ

ウインドウにはいくつかのプロパティが存在します。
プロパティはreadonlyです。直接値を変更しないでください。
画面との整合性がとれなくなります

ウインドウID
String
すべてのユーザーを通じてユニークな値で管理されます。
title

template
