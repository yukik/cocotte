データソースはイベントを発行します。

var ds = new Datasource(schema);
ds.on('add', function (row) {
    console.log('event - add');
});
リスナーの登録・解除は、node.jsのeventEmitterに準拠します。
発行されるイベントは、次の通りです。

add
listener ({Row} row)
行が追加された
remove
listener ({Row} row)
行が削除された
clear
listener ({Datasource} ds)
行がクリアされた
move
listener ({Datasource} ds)
現在行が移動された
change
listener ({Row} row)
いずれかの値が更新された
change {fieldName}
listener ({Row} row, {Mixed} value)
指定フィールドの値が更新された
currentChange
listener ({Row} row)
現在行の値が変更されたもしくは現在行が移動したことによって値が変更になった
currentChange {fieldName}
listener ({Row} row, {Mixed} value)
現在行の指定フィールドの値が変更されたもしくは現在行が移動する事によって指定のフィールドの値が変更になった
set default
listener ({Field} field)
フィールドの既定値が変更された
set filter
listener ({Field} field)
フィールドの条件が変更になった
