データソースの設定方法を説明します。 以下はモデルを元にして作成した最も単純な例です。 

var ds = {
    name : 'student'
  , model: 'student'
};
module.exports = exports = ds;

必須の項目はnameのみで、フィールドはモデルから取得します。
自動的に永続化の機能を付加します。

次の例ではフィールドを設定し、モデルにないフィールドを追加しました。
またモデルのフィールドのうち使用するものだけを含める事が出来ます。
studentモデルからはnameフィールドのみ設定されます。
scoreフィールドはstudentモデルには存在しないため、Datasourceのみで使用でき永続化の対象になりません。 

var ds = {
    name  : 'student'
  , model : 'student'
  , fields: {
                name : {}
              , score: {type: 'number', caption: '点数'}
            }
};
module.exports = exports = ds;

モデルを指定せずに作成されたデータソースは、永続化の機能を持ちません。
ただし、後述するダイアグラムを使用する事で、他のデータソースとの連携を行う事が出来ます。
永続化は必要ないが、連携を必要とする場面が発生した場合は作成することがあります
次の例ではモデルを設定していないデータソースです。 

var ds = {
    name  : 'student'
  , fields: {
                name : {type: 'text'  , caption: '氏名'}
              , score: {type: 'number', caption: '点数'}
            }
};
module.exports = exports = ds;

プロパティ

name
モデル名
caption
表示名
model
モデル
fields
フィールド
