コメント型は行へのコメント機能を追加します。
実装中のため、仕様できません


表示は時系列順に並びます。
変更できるかどうか？変更前のコメントの確認が出来るかどうか？
などの細かい設定ができます。

コメントにはステータスを設定する事が出来、ステータスを使用する事で
他のフィールドへの影響を追加し、承認ワークフローなどは実現できます


schema (共通)
	fieldType: {String}  データ型      固定値: 'comment'   protoにて設定
	name     : {String}  システム名    既定値: 'comment1'  protoにて設定
	caption  : {String}  表示名        既定値: 'コメント1' protoにて設定
	primary  : {Boolean} 主フィールド  既定値: false       protoにて設定
	autoTo   : {Function}自動設定      既定値: null        protoにて設定
	readonly : {Boolean} 読取専用      既定値: false       protoにて設定
	required : {Boolean} 必須          既定値: false       protoにて設定
	defaultTo: {Number}  既定値        既定値: null        protoにて設定
	filterTo : {Number}  条件          既定値: null        protoにて設定
