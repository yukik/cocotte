タグ型は、複数の任意の文字列を追加することができるフィールド型です。

実装中のため、仕様できません

選択型が固定した文字列を選択する事に対して、文字数が50文字以内であれば設定できます。 

/**
 * タグ型
 *
 * 文字列を複数設定出来る。
 * 重複して同じ文字列を一つの行には設定出来ない
 * タグ用の挙動を行う事をクライアントに持たせる事を意識して作成されている
 * モデルとバインドしている場合は自動的に使用されたタグ毎に登録数をカウントする。
 * datasource.getTags(fieldName, limit)で登録数が多い順にタグのリストと登録数を取得出来る
 * 
 * 文字列型と異なる点
 *     入力は必須に出来ない
 *     規定値・固定値は設定出来ない
 *     前後の空白は削除される
 *     改行は入力出来ない
 *     文字数制限は最小は設定なし、最大50文字に固定
 * 
 * 選択型と異なる点
 *     値リストを設定出来ない
 *     選択数を制限する事が出来る
 *     検索時に柔軟な絞り込みが出来る
 * 
 * schema
 *     なし
 * 
 * schema (binding時)
 *     filtering: 絞り込み予定値
 *     filtered : 現在の絞り込み
 */
