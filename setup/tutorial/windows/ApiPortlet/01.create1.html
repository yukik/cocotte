ポートレットは予め用意されているものを使用する事も出来ますが、独自に作成する事も出来ます。
ここでは最も簡単なポートレットを作成方法を行います。
今回作成するポートレットは、ウインドウスクリプトで作成した数をカウントするウインドウをそのままポートレット化します。
次の二つのウインドウを開いてみてください

チュートリアル - スクリプト
チュートリアル - ポートレット

どちらも同じ動作し、全く違いが無いように思われます。
しかし、このウインドウのindex.jsを比較すると、ポートレットを使用した場合と未使用の場合の大きな違いを確認出来ます

チュートリアル - スクリプト

ポートレットを使用していないウインドウです。
すべてのスクリプトをウインドウに設定した場合は、もちろん下記のindex.jsだけでは動作しません。
このほかにも、index.html, server.js, client.jsのファイルを作成する必要があります
実際のウインドウにスクリプトを作成する方法は、Window APIのスクリプト3で確認してください。 

var helper = require('cocotte/window/helper');
var win = {
    title   : 'チューリアル - スクリプト'
  , contents: helper.html(__dirname, 'index')
  , script  : require('./server')
};
module.exports = exports = win;

チュートリアル - ポートレット

ポートレットを使用していないウインドウです。
index.jsだけで目的のウインドウを表示する事が出来ます。
他にファイルを用意する必要はありません。
ここでは、Coutupという名前のポートレットを定義したことになっています。 

var win = {
	title   : 'チューリアル - ポートレット'
  , contents: { portlet: 'Countup' }
};
module.exports = exports = win;

上記のふたつのindex.jsを確認しました。
ポートレットを利用する事で劇的にウインドウの作成が楽になる事を確認する事が出来ます。
さらに、他のウインドウを作成した場合に、Countupポートレットは簡単に画面に追加する事ができます。
じつは、ウインドウに設定したスクリプトを、ポートレットに作成し直す事はそれほど難しくはありません。
