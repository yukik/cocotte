ログインとワークスペース

ログインは、ユーザーID/パスワードのほかに所属コードを入力する必要があります。
同じユーザーでも複数の所属を設定する事が出来ます。
ログイン後はワークスペースが表示されます。
これにより、業務内容が異なる作業をする場合に複数のワークスペースを切り替えて使用する事が出来ます。
また、所属によりアクセスできるデータが異なるように設定出来ます。
ワークスペースでは、ウインドウを開きデータを操作する事が出来ます。

ログアウト

初期のデザインでは、右下に扉のアイコンが配置されています。
ログアウトをする場合はアイコンをクリックしてください。

ウインドウの操作

ここでは既定のデザインでの操作説明をします。
開発者によりデザインが変更になった場合は、操作方法が変更になる場合があります

開く
ワークスペース左にあるドックのスタートバーをクリックします
スタートメニューが表示され、その中から目的のウインドウを開くリンクをクリックします
閉じる
ウインドウ右上の×のアイコンをクリックします。
閉じられたウインドウはすべての状態が破棄されます
サイズ変更
ウインドウの境界にマウスをあわせるとサイズを変更するアイコンに変わる部分でドラッグ＆ドロップします
右下はアイコンの変わる部分の領域がやや大きくなっているため、操作しやすくなっています。
移動
ウインドウのタイトルバーもしくはウインドウ下部のバーで移動のアイコン（十字）に変更する部分でドラッグ＆ドロップします
ウインドウを呼び寄せる
画面外に存在するウインドウを表示領域に呼び寄せる場合は、ドック内のミニバーをクリックします
手前に表示する
ウインドウの領域をクリックします
ウインドウ内にウインドウを開く為のリンクやボタンを設定します
最小化する
ウインドウ右上の＿（アンダーバー）のアイコンをクリックします。
最小化されたウインドウは、内容を保持したまま非表示になります。
最小化を解除する
ドック内のミニバーをクリックします。
最小化前の位置とサイズで表示されます。
最大化する
ウインドウ右上の■のアイコンをクリックします。
ウインドウを画面いっぱいに表示させます。
最大化を解除する
再びウインドウ右上の■のアイコンをクリックします。
最大化前の位置とサイズで表示されます
システムチェンジボードを開く
ウインドウ右下の！のアイコンをクリックします。
システムチェンジボードは利用者が開発者へ機能の変更を要望する際に使用します。
ヘルプを開く
ウインドウ右下の？のアイコンをクリックします。
ヘルプはウインドウの使用方法や注意すべき点を記載するためのウインドウを表示します。
このヘルプの内容は利用者が変更する事が出来ます。
ドックを手前に表示する
ウインドウの外をクリックします
ドックを最背面にする
再びウインドウの外をクリックします

コンテンツ

ウインドウ内に表示される情報をコンテンツと呼びます。
コンテンツの操作は、個別に作られたウインドウの設定とポートレットと呼ばれるモジュールによって異なります。
それぞれのコンテンツの操作方法を確認するにはヘルプをクリックしてください。
